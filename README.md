Repository dedicated to **legal IPTV resources** for Italian *free-to-air* TV channels.

## How to use the playlist

### 🔹 Option 1 — Direct link (recommended)

Copy and paste this link into your IPTV player: [link](https://github.com/mauro-midolo/IPTV-Italia/releases/latest/download/iptvitalia.m3u)

This method allows you to automatically receive playlist updates.

### 🔹 Option 2 — Download

1. Download the `iptvitalia.m3u` file from the root of this repository.
2. Open it with a compatible player (e.g. VLC).
3. Select the channel you want to watch.

## Disclaimer

This playlist includes only **free-to-air Italian TV channels** that are legally broadcast.  
No encrypted channels, pay TV services, or copyrighted content distributed without authorization are included.
